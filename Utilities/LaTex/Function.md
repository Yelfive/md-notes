# Function

## Equation alignment

Use `&=` to align equation, inside `\begin` block, this can be used only inside function environment.

```latex
\begin{aligned}
A &= B + C \\
  &= C + D + C \\
  &= 2C + D
\end{aligned}
```

Output:
$$
\begin{aligned}
A &= B + C \\
  &= C + D + C \\
  &= 2C + D
\end{aligned}
$$
