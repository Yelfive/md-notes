# Scope of C

## Scope of C Variables

### variable escape



```c
#include <stdio.h>

int *test() {
    int a = 1;
    printf("%p\n", &a);
    return &a;
}

int *test2() {
    int a = 2;
    printf("%p\n", &a);
    return &a;
}

int main(int argc, char *argv[]) {
    int *a = test();
    printf("a=%d\n", *a);
    test2();
    printf("a=%d\n", *a);
}
```
